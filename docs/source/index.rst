Welcome to TIFF–GDAL Docs!
==========================

**TIFF–GDAL Docs** is a guide and reference for working with TIFF images and GDAL-based
conversion and merging utilities in Python.

This documentation explains:
- How to read TIFF metadata and raster data.
- How to convert between file formats (GeoTIFF → NetCDF, etc.).
- How to merge multiple datasets efficiently.

Check out the :doc:`usage` section for examples, or learn more about
the :ref:`installation` and dependencies.
